Facial Recognition App

I developed a facial recognition app that can detect a face in an image using the Clarifai API. The app was created using the React javascript library and a backend developed wth Node.js and Express. The backend also links to a postgreSQL database which is used to store user login information. The app, server and the database were all deployed using Heroku.
